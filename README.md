# Investment Target Tracker

A high-performance, web-based financial tool designed to visualize compound interest and plan long-term wealth accumulation. This project serves as a strategic roadmap for consistent index fund investing (e.g., S&P 500) and passive income planning.

## Core Concept
The tool demonstrates the mathematical power of compound interest. It allows users to calculate the exact timeline required to acquire major assets (like real estate) by simulating monthly contributions with an annual growth step.

## Tech Stack
* **Frontend:** Vanilla JavaScript (ES6+), HTML5, CSS3.
* **Visualization:** [Chart.js](https://www.chartjs.org/) for dynamic, interactive data rendering.
* **Responsiveness:** Mobile-first approach using CSS Grid, Flexbox, and the `ResizeObserver` API.
* **Version Control:** Git & GitHub.

## Key Features
- **Compound Interest Logic:** Monthly compounding with precision-focused calculations.
- **Dynamic Contributions:** Unique feature to simulate annual salary/deposit increases (`Annual Growth`).
- **Real-time UX:** Instant updates and numerical "metamorphosis" animations upon input changes.
- **Adaptive Charts:** Fully responsive visualization that scales across Desktop, Tablet, and Mobile devices.
- **Clean UI:** Professional financial interface with localized currency formatting (€).
