# Investment Target Tracker

A web-based tool designed to visualize compound interest and help plan long-term financial goals. This project is a personal roadmap for wealth accumulation through consistent index fund investing

## Core Idea
The goal is to demonstrate the power of compound interest and calculate the exact timeline needed to acquire assets (like real estate) for passive income
